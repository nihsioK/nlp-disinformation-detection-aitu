2. Библиографический менеджер Zotero

Для организации источников использован менеджер Zotero. В библиотеке были созданы отдельные подборки по темам: классические методы машинного обучения, transformer-based models, context-aware approaches, диссертации и обзорные статьи. Такой подход позволяет быстро группировать источники по смыслу, экспортировать их в разные стили цитирования и поддерживать аккуратную структуру литературы по теме диссертации.
